### MCP + LSP（Language Service Protocol）

### Serena 的核心定位：一个赋能 LLM 的开源 IDE 工具集

#### 多层工具系统 (Multi-Layer Tool System)

• **全局符号搜索**：快速查找项目中特定名称的函数、类或变量

• **引用分析**：找出所有引用特定符号的代码位置

• **精准编辑**：直接替换整个符号定义或插入代码片段

• **上下文感知**：自动识别当前编辑位置的作用域

#### 语言服务器封装 (Language Server Wrapper)

#### 持久化记忆系统 (Memory System Architecture)
